git中使用https访问github经常会报错，改为使用ssh之后基本再出现过问题

使用ssh连接github具体过程和vscode配置免密登录过程差不多

1. 本地生成ssh的公钥和密钥

2. 将公钥(.pub)上传至服务器端

3. 配置登录时使用本地的密钥文件进行登录验证

   - 首先启动ssh-agent

       ```bash
       # start the ssh-agent in the background
       $ eval "$(ssh-agent -s)"
       > Agent pid 59566
       ```

   - 将密钥添加至ssh-agent

     ```bash
     $ ssh-add ~/.ssh/yourKeyFile
     ```
     

配置完成后即可在git中使用ssh访问远程仓库
